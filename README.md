# Multiplayer Asteroids with Node.js and p5.js

Welcome to the GitHub repository for our Multiplayer Asteroids Game, a modern take on the classic arcade game built with Node.js and p5.js. This project combines the simplicity of p5.js with the power of Node.js to create a real-time, multiplayer gaming experience.

## Features

- **Real-time Multiplayer Gameplay**: Players can join the game from different devices and compete or cooperate in real-time.
- **Node.js Backend**: Utilizes Node.js to handle multiple player connections, game logic, and state synchronization.
- **p5.js Graphics**: Leverages p5.js for rendering dynamic and responsive game visuals.
- **Responsive Design**: Adapts to various screen sizes to provide an engaging gaming experience on multiple devices.
- **Cross-Browser Compatibility**: Ensures a consistent experience across all modern browsers and phones.

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- Node.js - Download and install Node.js from [Node.js official website](https://nodejs.org/).

### Installation

1. Clone the repo:
  git clone https://github.com/paulduttonpad/AsteroidsJS.git

2. Navigate to the project directory:
  cd multiplayer-asteroids

3. Install NPM packages:
  npm install socket.io express matter-js node-p5
   
5. Start the server:
  node server.js

### Playing the Game
After starting the server, open your web browser and go to http://localhost:10000 to start playing. Use the arrow keys to move your spaceship and the spacebar to shoot asteroids.
